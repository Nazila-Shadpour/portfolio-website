CinematicWanders

Project Overview

The movie database was a project for the BCIT FWD Program. As part of a team project, I built a fully responsive Movie App using React and The Movie DB (TMDb) API. The app includes four main pages: Home, About, Individual Movie, and My Favourites. It showcases movies across categories like Popular, Top Rated, Now Playing, and Upcoming. Users can view detailed movie information, mark favourites, and have their favourite movies saved in local storage. The app is designed to work seamlessly on screen sizes ranging from 320px to 1980px, with a mobile-friendly navigation and clean UI.
