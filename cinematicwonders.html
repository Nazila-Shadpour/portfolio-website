CinematicWanders

Project Overview

The movie database was a project for the BCIT FWD Program. As part of a team project, I built a fully responsive Movie App using React and The Movie DB (TMDb) API. The app includes four main pages: Home, About, Individual Movie, and My Favourites. It showcases movies across categories like Popular, Top Rated, Now Playing, and Upcoming. Users can view detailed movie information, mark favourites, and have their favourite movies saved in local storage. The ability to add or remove favourites was implemented on the Home page, Individual Movie pages, and the Favourites page. The app is designed to work seamlessly on screen sizes ranging from 320px to 1980px, with a mobile-friendly navigation and clean UI.

Project Details

Design

This section covers the UI/UX design approach, layout planning, and responsiveness across devices. Our design focuses on a clean, user-friendly interface that effectively highlights movie content. In a previous class, we created a Hi-Fi wireframe interactive prototype for a movie app, which our team decided to use as the foundation for this project’s design phase. We aimed to implement the design in as much detail as possible, incorporating a dark theme and a movie theatre background image to reflect the cinematic experience. Intuitive navigation and a responsive layout were key priorities to ensure a smooth experience across all devices.

Development

The development phase focused on building the project using modern web technologies and a collaborative workflow. We used React to create modular, reusable components and manage dynamic state. Our team divided the work by pages, and I was responsible for developing the Individual Movie pages, where users can view detailed information and add or remove favourites. We integrated the TMDb API to fetch real-time movie data and used local storage to persist users’ favourite movies. Clean code structure, component-based architecture, and responsive styling were key priorities throughout development.

Reflection

This project challenged me to work with dynamic data and React components effectively. Working as part of a team was a valuable experience.

I learned how to collaborate, manage state, handle APIs, and build reusable components. One key challenge was ensuring responsive design across devices.

Moving forward, I’d like to improve the user experience by adding a search button to allow users to find movies by name.
